# 前端变更记录

## 当前版本

### 功能增强

- 接入真实后端接口：旅行计划生成 `/api/trips/plan`、知识库检索 `/api/knowledge/search`。
- 新增知识库检索调试台，支持查看请求体 JSON、响应体 JSON、命中文档、匹配分数和知识库来源。
- 支持可插拔目的地知识库展示，可区分四川知识库、云南知识库、通用旅行规则和 Mock 兜底。
- 新建行程表单支持自定义旅行偏好和避坑项。
- 结果页支持复制完整行程摘要。
- 每日详情页支持复制单日行程安排。
- 引用来源页支持复制单条知识库引用。
- 首页控制台改为动态展示最近生成的行程状态。

### 体验优化

- 增加空状态：未生成行程时，结果页、每日详情页、引用来源页会引导用户先新建行程。
- 增加生成结果的本地缓存，刷新页面后仍可查看最近一次行程。
- 增加清空并重新规划、重置表单能力。
- 增加后端接口状态提示，可区分真实接口连接和前端 Mock 兜底。
- 优化移动端底部导航文案和图标。
- 预算分析进度条改为根据后端返回数据动态计算。

### 稳定性

- 接口请求增加超时控制，规划接口 45 秒超时，知识库检索接口 12 秒超时。
- `sessionStorage` 读取增加 JSON 解析容错。
- 请求参数和行程结果增加结构标准化，避免缓存字段缺失导致页面异常。
- 后端不可用时，前端 Mock 行程和 Mock 知识库检索结果会跟随用户输入动态生成。

### 技术点

- Vue 3 + TypeScript + Vite + pnpm。
- Vue Router 管理页面路由。
- `sessionStorage` 保存最近一次请求和结果。
- 前端统一封装 API 请求、超时、fallback 和状态提示。
- 抽取工具函数管理请求复制、结果标准化、知识库来源解析和剪贴板复制。
